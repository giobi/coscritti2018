# Portale Genitori - Coscritti 2018

Sito web per i genitori della classe seconda elementare.

## Contenuti

- **Homepage** - Introduzione e ultimi aggiornamenti
- **Calendario** - Eventi e appuntamenti scolastici
- **Blog** - Comunicazioni e notizie
- **Info Utili** - Contatti, orari, informazioni pratiche
- **Dona** - Supporto alle attivit√† della classe tramite PayPal

## Tecnologie

- Jekyll (generatore di siti statici)
- GitHub Pages (hosting gratuito)
- Theme Minima (design minimale e responsive)

## Aggiornare il sito

Per aggiungere nuovi post al blog, creare un file nella directory `_posts/` con il formato:

```
YYYY-MM-DD-titolo-del-post.md
```

Esempio: `2025-11-16-comunicazione-importante.md`

## Configurazione PayPal

Per attivare le donazioni PayPal, modificare il file `dona.md` e sostituire `TUO_BUTTON_ID_PAYPAL` con l'ID del pulsante generato dal proprio account PayPal.

## Google Calendar

Per integrare Google Calendar, modificare `calendario.md` e inserire l'iframe embed fornito da Google Calendar.
